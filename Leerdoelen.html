Over mij

Leerdoel 1: Designs om kunnen zetten naar html en css.

Tijdens deze minor moest ik veel designen en ik heb daar nog moeite mee. Met de meesterproef wil ik deel zijn van het design proces, maar voornamelijk focussen op die designs kunnen vertalen naar html en css.

Hoe:

Dit doel wil ik gaan bereiken door mijn ideeën een beetje uit te werken, dit kan ik doen door een schets of een moodboard. Ook wil ik het design wat uiteindelijk is gemaakt goed aanhouden.

Dit wil ik bereiken door hulp te vragen aan mijn medestudenten en docenten.

Leerdoel 2: Hulp vragen aan docenten

Ik heb redelijk wat moeite met alle code en designen. Ik ben al meer gaan vragen, maar niet aan de docenten maar aan mijn klasgenoten. Voor simpele feedback is dit goed, maar ik heb ook heel vaak dat als ik daadwerkelijk een groot probleem heb ik niet naar de docent durf te stappen, en dit wil ik wel verbeteren.

Hoe:

Ik wil dit doel bereiken door vragen te stellen. Feedback aan mijn groepsgenoten vragen en als ik heel erg vast loop hulp vragen aan een docent.

Leerdoel 3: Leerdoel 3: GSAP leren

Wij krijgen dit vak 2 GSAP workshops aangeboden en mijn groepsgenoten willen dit gebruiken in ons project. Origineel was ik het niet van plan om te leren, maar met onze opdracht is dit het perfecte moment. Ik vond het animeren in het vak CSS to the rescue wel leuk en ik heb met HCD ook leuk kunnen animeren. Met het vak API heb ik al geëxperimenteerd met de javascript animation API en nu wil ik ook GSAP leren, zodat ik kan bepalen welke van de drie versies ik nou het fijnst vind om mee te werken.

Hoe:

Ik wil dit leerdoel bereiken door de tweede GSAP cursus te volgen en verschillende GSAP animaties te maken. Ik wil ook snappen hoe onderdelen van GSAP werken en die kunnen uitleggen.

Leerdoel 1: Designs om kunnen zetten naar html en css.

Ik heb mijn best gedaan vind ik. Aan het begin toen we nog niet echt wisten wat de opdracht was heb ik 2 ideeën bedacht. Toen we uiteindelijk een idee hadden, heb ik geprobeerd er aan mee te denken. Aangezien dit mijn leerdoel was, was het mijn taak om de code van het design te maken en ik vond dat redelijk lastig. Ik had het design gevold en vertaald, maar toen wist ik niet hoe ik verder moest. Ik heb toen aan de bel getrokken bij Vasilis, die eigenlijk zei dat dit een stap te ver is. Ik moet eerst designen leren voordat ik designen kan omzetten, ik weet niet of ik het daar compleet mee eens ben maar hij had een punt. De manier waarop ik nu werkte, werkte niet. Dus ik heb een stap terug gedaan en kreeg een boek van Vasilis over designen. Ik heb er doorheen gekeken en er is vast iets blijven hangen, maar zo'n boek is simpelweg te groot om in een week te leren. Ik heb met Joost het design verder uitgewerkt en daarin zag ik verschillende dingen die ik nooit heb gedaan, en ook nooit had bedacht dat het een ding was. Ik weet waar ik verder aan moet werken met mijn design leerproces wat al een betere stap is dan "ik wil het leren".

Leerdoel 2: Hulp vragen aan docenten

Ik denk dat ik dit leerdoel wel heb gehaald. Ik heb mijn best gedaan met vragen stellen aan de docenten en toen ik echt vastliep heb ik ook heel erg aan de bel getrokken bij Vasilis. Ik heb daar ook hele nuttige feedback uit kunnen halen en ik heb ook me best gedaan om daar iets mee te doen. Ik heb ook kleinere vragen gesteld, maar dat echt aan de bel trekken toen ik het niet meer zag zitten was een grote stap.

Leerdoel 3: Leerdoel 3: GSAP leren

Ik was origineel niet van plan dit leerdoel te hebben, maar ik vind wel dat ik er veel van heb geleerd. Ik heb dit jaar met 3 verschillende animatie manieren gewerkt en ik kan zeggen dat GSAP de moeilijkste is, maar wel de meeste opties aanbiedt. De code is sommige punten overzichtelijker dan met CSS, maar voor het level van mijn animaties was GSAP niet nodig geweest. De volgende keer zal ik dan ook beter overwegen welke van de 3 opties het beste is voor mijn plannen. CSS is goed genoeg voor de basis animaties, en terwijl GSAP meer opties geeft en die ook heel fijn waren om te gebruiken, denk ik dat het extra werk om het te laten werken in CSS alleen het met de meeste animaties waard zou zijn.
