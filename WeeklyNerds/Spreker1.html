Christian van Drunen

Kilian Valkhof

Ik was er niet bij, dus ik kan het gesprek niet samenvatten. Zie reflectie.

Ik was niet bij de eerste weekly nerd. Deze is gehouden door Kilian Valkhof. Wat ik uit de sites van andere mensen haal is dat het ging over verschillende onderdelen van de css codetaal, zoals de has selector. Ik denk dat dit wel een goede was om bij te wonen, maar op een later punt. Deze werd gehouden op dag 1 van de studie, dus ik denk niet dat ik er veel van zou hebben gesnapt. Als ik deze nu zou bijwonen had ik er wel informatie uit kunnen halen denk ik. Hij had het blijkbaar ook over een quote “Choose the least powerful language suitable for a given purpose”. Deze quote wordt op vele verschillende manieren gebruikt, dus ik weet niet in welke context hij het gebruikte, dus ik kan daar niet veel over zeggen. Hij heeft ook blijkbaar zijn eigen browser gemaakt, wat wel heel cool is naar mijn mening. Wat ik wel interessant vind is dat hij over dingen vertelde die ik ook uiteindelijk heb gebruikt, zoals de has selector. Maar ook page transitions en volgens mij heb ik ook een keer een dialog gebruikt, maar weet dat niet meer zeker.
